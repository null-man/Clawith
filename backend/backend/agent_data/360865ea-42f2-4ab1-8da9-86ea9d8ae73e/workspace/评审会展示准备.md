# 评审会展示准备 - 产品设计（低保真原型）

**主讲人**: 设计师-小设  
**评审会时间**: 2026-03-11 15:00

---

## 展示结构（约10分钟）

### 1. 产品定位回顾（1分钟）
- 产品名称：AI直播话术工具
- 目标用户：商家/运营人员（后台配置话术，分发给主播使用）
- 核心场景：商家准备好话术库，主播直播时直接用
- MVP核心功能：关键词输入 → 话术生成、模板库、一键复制、历史记录

---

### 2. 页面结构说明（1分钟）

```
主控台 → 话术生成页（核心）
     → 历史记录页
     → 设置页
```

---

### 3. 逐页展示设计思路（6分钟）

#### 3.1 主控台页面
**设计目标**：提供清晰的功能入口和快捷操作

**核心要素**：
- 顶部：品牌标识 + 用户信息 + 设置入口
- 主体：3个功能卡片
  - 快速生成话术（核心CTA，强调引导）
  - 话术模板库（辅助功能，预设模板）
  - 历史记录（便捷入口）
- 底部：今日统计数据（增强用户成就感）

**设计考量**：
- 功能分层清晰，主次分明
- 降低用户学习成本，第一眼知道能做什么

---

#### 3.2 话术生成页（核心功能）
**设计目标**：简洁高效的生成流程

**交互流程**：
1. 输入关键词 → 2. 选择场景 → 3. 选择风格 → 4. 生成 → 5. 结果操作

**核心要素**：
- 步骤式引导，降低输入门槛
- 场景选项：上新/促销/清仓/日常
- 风格选项：活泼/专业/沉稳
- 生成结果支持：一键复制/编辑/收藏

**设计考量**：
- 分步骤设计让复杂任务简单化
- 提供多种风格选择满足不同主播需求

---

#### 3.3 历史记录页
**设计目标**：方便用户管理和复用历史话术

**核心要素**：
- 搜索功能：快速找到历史记录
- 列表展示：显示关键词、场景、风格、时间、复制次数
- 操作按钮：查看/复制/删除

**设计考量**：
- 复制次数统计帮助用户识别高质量话术
- 支持搜索快速定位

---

#### 3.4 设置页
**设计目标**：个性化配置和账户管理

**核心要素**：
- 账户设置：头像、昵称、绑定账号
- 偏好设置：默认风格、话术长度
- 数据管理：导出/导入历史记录

---

### 4. 评审讨论点（2分钟）

请团队从以下角度提意见：
- **竞品视角**（小任）：差异化定位、功能设计
