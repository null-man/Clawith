## 1. 产品概述

| 项目 | 说明 |
|------|------|
| 产品名称 | AI直播话术工具 |
| 目标用户 | 商家/运营人员（后台配置话术，分发给主播使用） |
| 核心场景 | 商家准备好话术库，主播直播时直接用 |
| MVP核心功能 | 关键词输入 → 话术生成、模板库、一键复制、历史记录 |
